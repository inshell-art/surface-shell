# Changelog

## 0.1.0

- Created the `@inshell/surface-shell-core` workspace package.
- Added command-first and question-first parsing.
- Added command tree resolution, aliases, branch help, dispatch, redaction, history/transcript helpers, completions, events, in-flight locking, side-effect metadata, and plain-text rendering.
- Added THOUGHT-like and Ask-like standalone fixture validation.
- Added golden transcript tests and package-boundary checks.

The package remains private while the API is validated by real adopters.
